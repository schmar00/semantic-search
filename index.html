EXAMPLE: metadata catalogue search
instruction

This example is built to search the geocat.ch CSW catalogue web service. For the search logic is used the GEMET thesaurus hosted at the Geological Survey of Austria (Sparql endpoint).

To perform a semantic search via keywords - select a keyword promoted in the dropdown list.
The search passes through following steps (relevance criteria):
  1. exact matches in keywords
  2. full text matches in abstracts texts
  3. matches of keywords within a semantic distance of 1 (broader, narrower and related)
  4. matches of all narrower keywords
  5. full text matches of all narrower keywords in abstracts texts
If a term typed in and checked with Enter or Go! button - a full text search (language dependent!) is carried out.
For test purposes the number of results is cropped to a maximum of 3 each. In practice a search would stop if a total number of about 20 results is managed
The language of the keywords offered to search is choosen by the browser language.
